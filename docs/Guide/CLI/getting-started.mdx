## Installation

```bash npm2yarn2pnpm
npm install @sapphire/cli -g
```

:::info

You can install the CLI as a development dependency, but we recommend installing it globally so you can reuse the
utility in multiple projects.

:::

:::danger

CLI requires Node 16.7 or higher.

:::

## Creating a bot

Lets start creating a new project. The first thing to run is the `new` command:

```bash

```

use to generate your project.
